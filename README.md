# [Minerva](https://minerva-project.org/)
> **日本の政治を変える**

Minervaは、国政政治家の公約内容と、実行の有無を確認できるサイトです。
「ミネルヴァのフクロウは黄昏に飛び立つ」と言う言葉から名前を取りました。

## 参加方法
参加していただき本当にありがとうございます。
人間誰しも失敗をしますから、失敗を恐れずにぜひ参加してください。

選挙公報から公約を載せる際は  
> content/prefectures/都道府県/選挙区の番号/    

にてお願いします。  

選挙区.mdがテンプレートになっています。公約か否かの判断は**rubric.md**を元に。**rubric**をより改善する手を見つけた方は遠慮せずにPull requestしてください。      


政治家の選挙公報を追加する際は
>content/shu/都道府県/選挙区の番号/政治家の名前.md

にてお願いします。政治家.md　がテンプレートです。（テンプレートと言えるほどありませんが...）


## 理念

Minervaは、日本の国政政治家が掲げた公約の実行状況を、透明性の高いプロセスで客観的に評価するプラットフォームを目指しています。

### 客観性と中立性
>特定の政治団体への支持・不支持を目的としません。個人の主観的な努力や意図ではなく、公開されている情報に基づき何が成し遂げられたかという**客観的に検証可能な結果**のみに焦点を当てます。

### 議論を歓迎します！

> **懐疑心を忘れずに。**  あなたは他の誰とも異なる人生を歩んでいます。あなたの経験や視点は唯一無二のものです。疑問がある場合、あるいは改善案がある場合、ぜひあなたの考えを共有してください。




# なぜこのサイトが必要なのか。

選挙公報をめくれば、「検討」、「努力する」、「適切な対応」などの言葉が並びます。曖昧な公報からは政治家が何をしたいのか、何をするのか、有権者に伝わりません。
選挙公報に公約と呼ぶに値するものはほとんどありませんでした。

政治家は選挙になると盛んに国民の知性への負の信頼を披露します。
政策ではなく名前をひたすら繰り返し、景気の良いことばかり言う。
選挙が終われば、選挙での話はほとんど顧みられません。国民が政策の中身を理解せず、感情や雰囲気だけで投票すると考えているのでしょう。
無責任なポピュリズムで良いのか。日本はこのままで良いのか。国民はこんな政治でいいのか。

この現状を打破したい。　　

公約と実現の有無を簡単に確認できるウェブサイトがあれば公約が増えるのではないかと考え、作成しました。誠実に公約を掲げ実行する政治家が評価される土壌を作りたい。実現できなければ、政治家が自らの言葉で有権者に説明する責任を求める。一緒に「おまかせ民主主義」を少しでも変えませんか。  

あなたの力を貸してください。
