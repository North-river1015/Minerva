# [桜井 シュウ](/shu/hyogo/6/sakai-shu)

立憲民主党

✅ [教育と国家アイデンティティを護る。](日本の教育と国家アイデンティティを護り抜く。)  
✅ [国柄と国益を守る国家アイデンティティの確立。](日本の国柄と国益を守る国家アイデンティティの確立。)  
✅ [若者が未来の社会を動かす国へ。](若者が未来の社会を動かす国へ。)  
✅ [新しい教育の形を追求する。](既存の教育システムに疑問を持ち、新しい教育の形を追求するために。)  
✅ [歳 壊さ れる 日本 の 教育 と 国家 アイデンティティ を 護り 抜く 。 日本 の 国柄 と 国益 を 守る 国家 アイデンティティ の 確立 。 若者 が 未来 の 社会 を 動かす 国 へ 。](歳 壊さ れる 日本 の 教育 と 国家 アイデンティティ を 護り 抜く 。 日本 の 国柄 と 国益 を 守る 国家 アイデンティティ の 確立 。 若者 が 未来 の 社会 を 動かす 国 へ 。)  
✅ [児 の 父 。2 社 経営 。 既存 の 教育 システム に 疑問 を 持ち 、 新しい 教育 の 形 を 追求 する ため に 淡路島 に 移住 。 子ども を 自然 型 フリー スクール に 通わ せ つつ 、 より よい 教育 の あり方 を 模 索 中 。 仕事 は WEB 広告 関連 の サービス 提供 と ホテル 再生 事業 。](児 の 父 。2 社 経営 。 既存 の 教育 システム に 疑問 を 持ち 、 新しい 教育 の 形 を 追求 する ため に 淡路島 に 移住 。 子ども を 自然 型 フリー スクール に 通わ せ つつ 、 より よい 教育 の あり方 を 模 索 中 。 仕事 は WEB 広告 関連 の サービス 提供 と ホテル 再生 事業 。)  
✅ [歳 から の 投票 権 。](歳 から の 投票 権 。)
